‚ö° Core Systems Cheat Sheet

30-Day Deadline: You have 30 in-game days to reignite all 9 Azure Monuments and save the world. Main Story quests do NOT consume time!
Action Points (AP): You have 3 AP per day. Sleeping restores AP but advances time by 1 day. Use AP wisely!
One-Day Main Quest: Character story quests happen in a single day but don't consume deadline time. You'll be locked to that area until complete.

üí° Strategic Tips

üéØ Prioritization

Focus on one kingdom at a time. Complete character arcs before moving to the next region for maximum story coherence.

‚è∞ Time Management

Don't panic about the 30-day deadline! Main quests don't consume time. Use your days for exploration and relationship building.

üí¨ Character Choice

You can only focus on ONE character per city. Choose based on your preferred class or story theme. Other characters continue off-screen.

üó∫Ô∏è Monument Order

While you can complete monuments in any order, consider your party composition. Get support classes (Synergist) early for easier progression.

üé≠ Replayability

The game is designed for multiple playthroughs. Each character path reveals different aspects of the story. Try different combinations!

‚ö° AP Efficiency

Plan your 3 AP uses each day. Balance combat progression (dungeons) with social activities (Hang Out) for optimal growth.

üîÆ Secrets & Unlockables

New Game+

After completing the game once, you unlock NG+ mode. This allows you to:

  • Start with all Azure Resonance from previous playthrough
  • Access hidden character conversations
  • Unlock true ending if all 12 character arcs completed across playthroughs

Endless Tower

Unlocks after completing any 3 Azure Monuments. Procedurally generated challenge tower with exclusive rewards. Perfect for grinding materials and testing builds!

Secret Ending Requirements

To unlock the true ending:

  • Complete all 12 character arcs (across multiple playthroughs)
  • Light all 9 Azure Monuments
  • Reach Babel Tower with all heroes' memories collected
  • Make the "third choice" when confronting GoD
